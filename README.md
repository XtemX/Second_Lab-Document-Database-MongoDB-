# Бекенд для блогу на базі MongoDB & Flask

Цей проєкт є лабораторною роботою, що демонструє розробку веб-бекенду з використанням Flask та NoSQL бази даних MongoDB (PyMongo).

## Реалізований функціонал:
- **Аутентифікація користувачів:** Реєстрація та логін із хешуванням паролів (`werkzeug.security`).
- **Керування контентом:** Створення та перегляд статей за категоріями, додавання коментарів.
- **Індексація (Вимога ТЗ):** - Створено унікальний індекс для `email` у колекції користувачів.
  - Створено індекс для швидкого сортування статей за датою (`created_at`).
  - Створено індекс для оптимізації пошуку коментарів за `post_id`.

## Як запустити:
1. Переконайтеся, що MongoDB запущено на `localhost:27017`.
2. Встановіть залежності: `pip install flask pymongo werkzeug`
3. Запустіть додаток: `python lb2.py`
